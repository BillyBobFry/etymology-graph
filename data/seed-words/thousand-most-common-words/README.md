# Thousand Most Common Words Seed Data

Normalized seed-word files derived from [SMenigat/thousand-most-common-words](https://github.com/SMenigat/thousand-most-common-words).

- License: MIT; see `LICENSE.md`.
- Upstream data source listed by that project: <https://1000mostcommonwords.com/>.
- Retrieved: 2026-05-28.
- Files: 98 language JSON files.
- Words: 98000.

Each JSON file uses the app seed-list shape consumed by `pnpm seed:extract:popular`: language metadata, source metadata, and a `words` array.
